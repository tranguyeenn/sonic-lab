# About This Repository
Ruby is a programming language that is used to run Sonic Pi, which is a live coding music system. This language is very beginner friendly and anyone can learn if want to.

# How to Run Code
1. Go to [sonic-pi](https://sonic-pi.net/) and download based on your OS.
2. Copy and paste my code over to the code editor.
3. Click run
4. If it is a live loop, click stop whenever you want. It will not stop until you click stop.
